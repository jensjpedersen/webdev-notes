

# Shortform operators
* num++ - num = num + 1
* num-- - num = num - 1
* num+=5 - num = num + 5
* num-=5 - num = num - 5
* num*=5 - num = num * 5

`float.toFixed(2) // 2 decimals` 

* Set number of total digits - returns a string.
