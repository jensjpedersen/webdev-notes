
# Object 

## Syntax: 
```javascript
var obj = {
    key1: value1,
    key2: value2,
    key3: value3
}
```

### Shorthand
* Can use variables as keys and values.
* `key#` will be the name of the variable and `value#` will be the value of the
  variable. 
```javascript
var obj = {
    key1,
    key2,
    key3
}
```
